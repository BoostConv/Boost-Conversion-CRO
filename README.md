# generate-lp — Landing page e-commerce haute conversion pour Claude Code

> **La méthode qui fait scaler les meilleures marques e-commerce (DTC) françaises, maintenant dans ton terminal.**

**Un seul skill, 100% autonome, dédié au e-commerce.** Tu décris ton produit, Claude te
génère une **landing page e-commerce haute conversion** (page de vente produit) dans un
seul fichier HTML — aucune dépendance, aucun compte, aucune clé API. Ouvrable directement
dans un navigateur, prête à brancher sur ta boutique Shopify, WooCommerce ou autre.

Ce n'est pas un générateur de templates de plus. Chaque page est construite avec la
**Méthode Océan Vert** de [Boost Conversion](https://www.boostconversion.fr) — le
framework de conversion forgé au contact de marques qui ont fait du chiffre, pas de la théorie :

### Éprouvée avec des marques e-commerce qui scalent

**Spring · 900.care · Inga · Gaya** *(et d'autres marques DTC e-commerce à forte croissance)*

Ces marques ne vendent pas par hasard : sur leurs pages produit, elles maîtrisent le
**niveau de conscience** de leur audience, nomment **l'ennemi** de leur client, et
construisent une **offre irrésistible** avant la première ligne de copy. C'est exactement
ce que ce skill applique automatiquement à tes propres pages produit — le même raisonnement
stratégique e-commerce, pas juste du joli HTML.

## Ce que fait le skill (ce qui te fait dire "waouh")

- **Diagnostic stratégique** affiché avant d'écrire — niveau de conscience (pyramide Schwartz), structure de page raisonnée, angle d'accroche. Tu vois la stratégie, pas juste le résultat.
- **Copy e-commerce qui vend** : hero qui nomme l'ennemi du client, agitation, preuve sociale, offre produit (prix, bonus, livraison), garantie, FAQ qui désamorce les vraies objections d'achat.
- **Design premium par défaut** : design system complet (typo fluide, hero soigné, micro-animations sobres au scroll), responsive, CTA sticky mobile. Un rendu dont tu n'auras pas honte.
- **Sortie = 1 fichier `.html` autoportant** : zéro build, zéro dépendance, hébergeable partout en 30 secondes.

```

```
